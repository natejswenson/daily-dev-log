# Daily Dev Log

Daily development log entries for personal projects. Entries are markdown files organized by project, consumed by [natejswenson.io](https://natejswenson.com/devlog).

## Structure

```
<project>/
  manifest.json    # Index of all entries
  YYYY-MM-DD.md    # Daily entry
```
